if分とは
・条件や状況に応じて実行内容を変えたい場合に使用する
→「もしAならばBを実行し、AでなければCを実行」のような時
・真偽判定を行う
　条件に当てはまればtrue
　条件に当てはまらなければfalse
　→ 条件に当てはまる場合はif 〜 end 間に記述されている処理が実行される

	apple = 110
	if apple >= 100
		puts "りんごの値段は100円以上です"
	end

	if apple <= 100
		puts "りんごの値段は100円以下です"
	end						→	りんごの値段は100円以上です

比較演算子
・条件を比較して判定をするもの

比較演算子種類
・<：右辺の方が大きい
・<=：右辺の方が大きい、または等しい
・>：左辺の方が大きい
・>=：左辺の方が大きい、または等しい
・==：右辺と左辺が等しい	← 文字列の比較でも利用可能
・!=：右辺と左辺が等しくない	← 文字列の比較でも利用可能
ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー

論理演算子
・2つ以上の条件を判定する際に使用する

論理演算子の種類
・!：否定
・&&：どちらの条件も正しければtrue
・||：どちらかの条件が正しければtrue

	tall = 180
	if tall >=170 && tall <= 190
		puts "身長は170以上190以下です。"
	end						→	身長は170以上190以下です。

elsif, else
if文では、elsifやelseを使うことで、ifの条件に当てはまらなかった場合の処理を設定できる

	apple = "Aomori"
	if apple == "Nagano"
		puts "りんごの生産地は長野です"
	else
		puts "りんごの生産地は青森です"
	end					→	りんごの生産地は青森です


	apple = "Yamagata"
	if apple == "Nagano"
		puts "りんごの生産地は長野です"
	elsif apple == "Yamagata"
		puts "りんごの生産地は山形です"
	else
		puts "りんごの生産地は青森です"
	end					→	りんごの生産地は山形です
